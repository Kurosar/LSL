RealFire by Rene

Menu-controlled realistic particle fire with smoke, light and sound.
Features:
	Fire with smoke, light and sound
	Burns down at any desired speed, looped or non-looped
	Change size, color and sound volume
	Access control: public, group, owner, group+owner
	Store default settings in notecard
	Plugin support: call the menu from other scripts
	Add your own extension button
	Add more switches to the same fire but never get out of sync
	Remote control: external switch that works across an entire sim
	Control panel for up to 9 fires
	Multiple fires using one


Please note that the user manual is from the website. The site itself is no longer active.

There may be references to examples that no longer exist.
