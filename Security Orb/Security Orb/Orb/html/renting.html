Renting Land and Apartments

Rented Land and Apartments

Since version 9.008 the Prim can detect land borders. This allows landlords to have HS Prims protect one parcel of land and not affect the neighbours parcel.

The setting to enable this is called Other Land Safe and it is found in the menus by going to SETTINGS / Weapon / OtherLand.

It is also settable for experienced users by editing the "HomeSecurity Settings" notecard and changing: other_land_safe=1 then using the command: /8reset

 /8reset command will reset the Prim to the Settings notecard defaults but names won't be lost.


This Settings notecard feature can be used for:
  • copying one Settings notecard into multiple Prims, then /8reset them, for faster set ups.
  • /8reset could also be used to reset the Prim after a renter leaves (Names will need to be removed via menus).

Apartment Complexes

You need to update to HS Prim version 9.010 or above before following these instructions.

Appartments have an additional problems caused by the fact that each appartment may have neighbouring appartments above, below and each side of the appartment.

The HomeSecurity™ Prim solves these issues with two methods working together:

  1. The first is commands to tell the HS Prim how far above itself the roof is and how far below it the floor is. These commands are:

    /8up=number (where number is above 0.0)

    and

    /8down=number (where number is above 0.0)

    The number can be anything including decimal point numbers.
    The Prims attack range should be "ThisLand".


  2. The second method is designed to tell the HS Prim where the outer walls of the apartment is. Rather than use coordinates which can be very technical and confusing or involve extra prims the Prim uses land borders. This means that each column of apartments should be on different land.     Once each column has its own unique land name then the HS Prim needs the setting "Other Land Safe" ON. This can be done through the normal set up or in SETTINGS / Weapon / OtherLand.


    		 Each Prim needs to be more than 20m apart or menus will show from each HS Prim within 20m. If the HS Prims can not be placed more than 20m apart make each of the HS Prims use a different channel to the others within 20m. You do this by editing the HomeSecurity Channel notecard inside the HS Prim and changing channel=8 to another number. Each HS Prim within 20m of another will need a different number.
