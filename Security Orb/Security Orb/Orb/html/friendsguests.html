  • Guests are the same as the old Authorised avatars.
  • Guest Mode is the same as the old Private Mode.
  • Version 10 Authorised avatars will be made into Version 11 Guests during Version 11 upgrade.
  • To use the Prim in the old Private Mode way just add people as Guests and use Guest Mode.
  • Warning: If you choose Friend Mode your Guests will be attacked, to avoid this choose Guest Mode.

New Features Added By Using the Friend/Guest Concept

  • A Friend is a brand new idea for Version 11+
  • Friends Mode is a brand new idea for Version 11+
  • Friends are always safe.
  • Friends Mode ONLY allows Friends and the Owner.
  • Friends Mode disallows Guests and GroupSafe members1 without the need to delete them.
  • You can swap between Guests Mode and Friends Mode to allow/disallow Guests and GroupSafe members at will.
  • You can view Friends and Guests by clicking Names / Visitors on the HS Prims menu. Names in capitals are friends, lower case are guests.
  • This feature allows you extra privacy (lovers, family, business meeting etc.) when you need it and more relaxed privacy later when you change back to Guest Mode.
